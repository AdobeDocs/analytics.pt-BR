# Devoluções

Uma visita que consiste em uma chamada única de servidor. Por exemplo, uma visita de página única é uma rejeição se o visitante não interagir com a página de forma a enviar dados para a Adobe como, por exemplo, clicar em um link ou iniciar um vídeo. Se mais um hit for recebido em uma visita, uma Rejeição não é contabilizada.
