# Idioma

Captura a configuração de idioma no navegador do visitante, permitindo que você veja a receita e os visitantes exclusivos relacionados à configuração de idioma. Quando você sabe o efeito que os idiomas preferidos têm nas métricas de sucesso do site, pode decidir como apresentar seu site em outros idiomas. Por exemplo, se você notar que um grande número de clientes de língua alemã geram atividade de carrinho de compras em seu site, você pode traduzir seu site para esse idioma e implementar uma campanha de marketing para direcionar o tráfego para seu site em alemão.

**[!UICONTROL Visitor Profile]** > **[!UICONTROL Languages]**

Preenchemos esse relatório com base na porção de idioma de aceitação da solicitação de cabeçalho http. O cabeçalho HTTP de idioma de aceitação é configurado como parte da configuração de linguagem de preferência de cada navegador:

[https://www.w3.org/International/questions/qa-lang-priorities.en.php](https://www.w3.org/International/questions/qa-lang-priorities.en.php)

Se um navegador não tiver um idioma especificado, ou se o navegador retornar uma ID de idioma=0, o relatório mostrará &quot;Não especificado&quot; no gráfico. O suporte do navegador para essa configuração pode afetar o número de retornos &quot;Não especificado&quot;.

Alguns itens de linha neste relatório incluem regiões. Essas são subtags de região.

>[!NOTE] A granularidade de hora em hora não é compatível.
