# Usar funções

As funções permitem filtrar/classificar seus dados e realizar análises estatísticas.

Para obter uma lista com todas as funções, consulte Funções [](../../../../../components/c-calcmetrics/cm-reference/cm-functions.md#concept_E3022D5EEEE145B69A23438BAF7016B2) básicas e funções [](../../../../../components/c-calcmetrics/cm-reference/cm-adv-functions.md#concept_A5FB9127D70F4E1AA02D1ACBF4F54174)avançadas.

> [!NOTE] Ao incluir funções em uma fórmula de métricas calculadas, sempre aplique a função antes de arrastar métricas ou segmentos.
