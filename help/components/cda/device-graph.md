# Gráfico de dispositivos

>[!WARNING]
>
>O Gráfico de dispositivos na Análise entre dispositivos foi [descontinuado](https://experienceleague.adobe.com/en/docs/discontinued/using/device-graph) e não está mais disponível a partir de **31 de dezembro de 2025**. Alterne qualquer conjunto de relatórios virtuais habilitado para o gráfico de dispositivos atual para o [método baseado em campo](/help/components/cda/field-based-stitching.md).
>
