# Importação de navegador e FTP

Depois de copiar os dados adequados para o arquivo modelo de classificações, você pode importar (upload) para os servidores de coleção de dados da Adobe.

>[!NOTE] Não é necessário fazer download (usando o recurso de exportação) de seus dados de classificação existentes antes de carregar os novos dados. Você só precisa carregar um novo arquivo contendo as novas linhas de dados. Este método melhora o desempenho.

See [General File Structure](/help/components/c-classifications2/c-classifications-importer/c-saint-data-files.md) for important information about the structure and limitations of uploaded files.

Consulte [Problemas comuns no upload de classificação](https://helpx.adobe.com/br/analytics/kb/common-saint-upload-issues.html) para obter ajuda com a resolução de problemas.
