# Análise unificada - a nova página de aterrissagem do Adobe Analytics

O Unified Analytics é uma nova página de aterrissagem para Análise de Adobe. Ele reúne o Analysis Workspace e o Reports &amp; Analytics em uma única interface e ponto de acesso sob o guarda-chuva do Workspace. Ele apresenta uma nova página inicial do gerente de projeto, um menu de relatórios atualizado e relatórios modernizados, bem como uma nova seção de aprendizado que ajuda você a começar de forma mais eficaz.

>No momento, essa página está na versão beta aberta e é um ponto de partida opcional. Se preferir, você pode usar sua landing page existente, seja Workspace ou Relatórios.
