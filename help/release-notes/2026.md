# Notas de versão de 2026 do Adobe Analytics

Saiba mais sobre as atualizações de versão mais recentes para [produtos Adobe Experience Cloud](https://business.adobe.com/br/products/adobe-experience-cloud-products.html). Obtenha a documentação de autoajuda, os tutoriais e os cursos mais recentes da Experience League.

## Fevereiro de 2026 {#feb26}

| Recurso e descrição | [Início da implantação](releases.md) | [Disponibilidade geral](releases.md) |
| ----------- | ------- | ---- |
| **Feeds de dados aprimorados** <br/>As seguintes melhorias agora estão disponíveis para feeds de dados:<ul><li>Experiência do usuário aprimorada.</li><li>Nova experiência para criar e gerenciar modelos de coluna.</li><li>Agora você pode escolher quando criar um modelo de coluna para reutilização em feeds de dados futuros. Também é possível excluir, editar e copiar modelos.<br/>Anteriormente, cada feed de dados criado resultava em um novo modelo de coluna, que resultava em um grande número de modelos de coluna não utilizados e não era possível excluí-los ou gerenciá-los.</li><li>Adicione e filtre por tags.</li><li>Exibir o histórico de trabalhos do feed de dados. (Quando o período de solicitação começou, terminou e assim por diante.)</li><li>Reenviar ou reprocessar feeds de dados. (Na página Histórico de trabalhos)</li><li>Permitir ocorrências de chegada tardia. Agora é possível incluir dados que chegaram depois que o trabalho do feed de dados terminou de processar dados dentro da frequência de relatório definida.</li><li>Ao escolher uma data final para um feed de dados, a data final escolhida é incluída como o último dia do feed de dados.<br/>Anteriormente, a data final era excluída do feed de dados.</li><li>Uma frequência de exportação de 15 minutos agora é possível, mas não está disponível por padrão. Para que essa opção fique disponível em seu ambiente, primeiro entre em contato com o Atendimento ao cliente da Adobe e solicite que seu conjunto de relatórios faça exportações a cada 15 minutos.</li></ul><p>**Observação:** com essas melhorias, os URLs das páginas de feed de dados no Adobe Analytics também serão atualizados. Atualize todos os marcadores existentes para apontar para as novas páginas de feeds de dados até 30 de abril de 2026.</p><p>Para obter mais informações, consulte [Criar um feed de dados](/help/export/analytics-data-feed/create-feed.md) e [Gerenciar feeds de dados](/help/export/analytics-data-feed/df-manage-feeds.md).</p> | quarta-feira, 20 de janeiro de 2026 | quarta-feira, 24 de fevereiro de 2026<p>(Planejado originalmente para lançamento em quarta-feira, 10 de fevereiro de 2026)</p> |
| **Novo Guia da API dos Conjuntos de Relatórios do Adobe Analytics 2.0** <p>Criar, recuperar, atualizar e excluir APIs do conjunto de relatórios. Para obter mais informações, consulte o [guia de referência da API do Conjunto de relatórios do Adobe Analytics 2.0](https://developer.adobe.com/analytics-apis/docs/2.0/apis/report-suites/) e o [guia de ponto de extremidade da API do Conjunto de relatórios do Adobe Analytics 2.0](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/report-suites/).</p> | | Fevereiro de 2026 |
| **Novo Guia de relatório de detecção de anomalias da API do Adobe Analytics 2.0** <p>Criar um relatório de detecção automática de anomalias da API. Para obter mais informações, consulte o [Guia de ponto de extremidade do Relatório da API de Detecção de Anomalia](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/reports/anomaly/).</p> | | Fevereiro de 2026 |
| **Novo Guia de Relatório de Tendências de Data Básica da API do Adobe Analytics 2.0** <p>Criar um relatório automatizado de KPI de Tendência de Data Básica de API Para obter mais informações, consulte [Guia de ponto de extremidade do Relatório de API de Detecção de Anomalia](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/reports/kpi/).</p> | | Fevereiro de 2026 |
