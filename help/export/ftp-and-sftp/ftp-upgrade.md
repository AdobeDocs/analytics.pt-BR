# Atualizar servidores FTP e SFTP da Adobe

As atualizações para servidores FTP e SFTP da Adobe podem apresentar novas configurações que afetam scripts de automação (geralmente usados para baixar ou carregar dados regularmente), que geralmente são criados para obedecer determinadas configurações de servidor.

Por exemplo, um novo status de sucesso é apresentado e afeta um script que usa o status de sucesso para acionar uma determinada ação. A Adobe notifica proativamente os usuários sobre essas alterações de configuração. Se a Adobe notificar que uma atualização do servidor FTP está iminente, verifique seus scripts de automação para garantir que eles ainda estejam funcionando corretamente.
