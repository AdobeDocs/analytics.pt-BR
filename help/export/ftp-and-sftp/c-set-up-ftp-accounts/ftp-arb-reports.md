# Relatórios entregues do Report Builder

Use o complemento Adobe Report Builder para o Microsoft Excel e crie relatórios personalizados para dados de análise da Web que podem ser inseridos em planilhas do Excel.

As solicitações podem fazer referência, de forma dinâmica, a células da planilha, e você pode atualizar e personalizar o modo como os dados são apresentados. Para FTP, você precisa garantir que o host esteja disponível externamente. Para obter mais informações sobre o [!UICONTROL Report Builder] relatórios entregues, consulte a [Ajuda do Report Builder](https://experienceleague.adobe.com/docs/analytics/analyze/report-builder/home.html?lang=pt-BR).

Para obter informações sobre limites FTP e retenção de dados, consulte [Limites FTP e retenção de dados](/help/export/ftp-and-sftp/ftp-limits.md).
