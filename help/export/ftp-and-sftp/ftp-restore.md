# Restaurar contas e dados FTP ou SFTP excluídos

Os dados excluídos de uma conta FTP ou SFTP não podem ser restaurados pela Adobe.

Se uma conta FTP ou SFTP que você pretendia usar tiver sido desativada, entre em contato com a equipe de conta da Adobe, que poderá discutir opções adicionais com você.
