# Métricas Padrão:

O Analytics exibe um conjunto padrão de métricas em todos os relatórios e projetos, a menos que um usuário selecione um conjunto personalizado de métricas. As métricas selecionadas são exibidas para todos os usuários do conjunto de relatórios associado. É possível atualizar uma métrica padrão do conjunto de relatórios de cada vez. Essas configurações não garantem a ordem em que as métricas padrão são exibidas.
