# Descrições do relatório personalizado

Você pode fornecer descrições de relatórios personalizados para eVars, props, classificações e eventos. As descrições estão visíveis para todos os usuários finais.

**[!UICONTROL Analytics]** > **[!UICONTROL Admin]** > **[!UICONTROL Report Suites]** > **[!UICONTROL Edit Settings]** > **[!UICONTROL *variável *]**

>[!NOTE] A adição de descrição está disponível para a maioria das classificações tradicionais, porém não está disponível para classificações móveis.

![](assets/report_descriptions.png)

1. Forneça uma descrição para o relatório. O número máximo de caracteres é 255.
1. Clique em **[!UICONTROL Save]**.

A descrição é visível em dois lugares:

* Na barra de título do relatório:

   ![](assets/report_description_2.png)

* Como uma dica de ferramenta quando você passa o mouse sobre o relatório na navegação à esquerda:

   ![](assets/report_description_3.png)
