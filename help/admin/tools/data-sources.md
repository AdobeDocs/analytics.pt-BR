# Fontes de dados

O recurso Fontes de dados permite importar dados para o Analytics de fontes offline. Após a importação, esses dados podem ser tratados e tratados de forma semelhante aos dados que foram coletados nativamente.

Use as Fontes de dados para:

* Integre dados offline com dados coletados do seu site.
* Coletar dados de servidores da Web, call centers ou outros sistemas. Em seguida, importe-o para o CX Enterprise para ver a conversão de todos os caminhos da sua organização.
* Otimizar o gerenciamento de ofertas de palavras-chave com base no preenchimento offline de clientes potenciais por meio do SearchCenter.

Quando as Fontes de dados são ativadas em um conjunto de relatórios, é fornecido um local FTP para que você possa carregar arquivos a fim de integrar aos relatórios. Após feito o upload, os servidores da Adobe processam automaticamente os dados e os integram ao conjunto de relatórios.

Consulte [Ajuda das Fontes de Dados](/help/import/data-sources/overview.md).
