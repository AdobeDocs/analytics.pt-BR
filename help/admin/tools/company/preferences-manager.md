# Gerenciador de preferências

A página Gerenciador de preferências permite configurar como renderizar saídas de relatórios do Excel e com valores separados por vírgula (CSV).

**[!UICONTROL Analytics]** > **[!UICONTROL Administrador]** > **[!UICONTROL Todos os administradores]** > **[!UICONTROL Configurações da empresa]** > **[!UICONTROL Preferências]**

* Usar formato de data obsoleto para CSV (30 de agosto de 20XX)
* Use o formato de data ISO 8601 para CSV (20XX-08-30)
