# Relatórios do Report Builder

Gerenciar a licença atribuída aos usuários do Report Builder.

Cada licença do Report Builder atribuída a um usuário recebe automaticamente dez tarefas programadas. Esse número de tarefas pode ser ajustado e realocado. Tarefas agendadas não podem exceder o número de tarefas disponíveis para cada usuário. Tarefas com excesso de agendamento não são executadas.

É possível conceder acesso ao Report Builder a qualquer usuário do [!DNL Adobe Analytics] gerenciando perfis de produtos no [Adobe Admin Console](/help/admin/admin-console/home.md).
