# Descrições do relatório personalizado

Você pode fornecer descrições de relatórios personalizados para eVars, props, classificações e eventos. As descrições estão visíveis para todos os usuários finais.

**[!UICONTROL Analytics]** > **[!UICONTROL Administração]** > **[!UICONTROL Conjuntos de relatórios]** > **[!UICONTROL Editar configurações]** > **[!UICONTROL *Canais de marketing *]**.

>[!NOTE]
>
>A adição de descrição está disponível para a maioria das classificações tradicionais, porém não está disponível para classificações móveis.

![](/help/admin/tools/assets/report_descriptions.png)

1. Forneça uma descrição para o relatório. O número máximo de caracteres é 255.
1. Clique em **[!UICONTROL Salvar]**.

A descrição é visível em dois lugares:

* Na barra de título desse relatório:

  ![](/help/admin/tools/assets/report_description_2.png)

* Como uma dica de ferramenta ao passar o mouse sobre o relatório na navegação à esquerda:

  ![](/help/admin/tools/assets/report_description_3.png)
