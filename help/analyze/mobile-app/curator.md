# Tarefas do curador

Os curadores podem ser administradores organizacionais ou pessoas em outras funções responsáveis pela configuração de scorecards para aplicativos. Os scorecards permitem que os usuários executivos visualizem uma ampla renderização de dados resumidos importantes de forma rápida e fácil em seus próprios dispositivos móveis. Usuários executivos são os usuários finais dos painéis do Analytics. Como curador, você deve concluir essas tarefas para configurar efetivamente o aplicativo para executivos:

1. [Crie scorecards para executivos](/help/analyze/mobile-app/create-scorecard.md).
1. [Assista os executivos com a configuração do aplicativo e usando o aplicativo](/help/analyze/mobile-app/set-up-execs.md).
