# Salvar uma pasta de trabalho com solicitações

{{legacy-arb}}

Ao criar relatórios com solicitações inseridas, você pode salvá-los usando **Arquivo** > **Salvar** ou **Arquivo** > **Salvar como** no Excel. O Report Builder detecta se o relatório contém solicitações. Ao selecionar uma das opções de gravação, conclua o formulário **Salvar Pasta de Trabalho como**.

* Como prática recomendada para qualquer trabalho extenso com aplicativos do Windows, a Adobe recomenda salvar suas solicitações na planilha com frequência e regularidade para evitar uma perda inesperada de solicitações em sua planilha.
* Ao nomear sua pasta de trabalho, considere usar um número de versão no nome do arquivo para que você possa preservar um histórico de trabalho. Por exemplo, nomeie sua primeira pasta de trabalho [!DNL web_forecast_01_01.xlsx].
* Se você já tiver salvo o relatório, o formulário [!UICONTROL Salvar modelo] não será exibido quando o relatório for salvo uma segunda vez. Se o relatório não contiver solicitações, essa caixa de diálogo não será exibida. Em vez disso, o formulário padrão [!UICONTROL Salvar como] do Excel é exibido.

## Nomes de arquivos e local {#section_2406629E9B644CE08430826948977D5D}

O formulário [!UICONTROL Salvar modelo] tem algumas das mesmas funções que a caixa de diálogo [!UICONTROL Arquivo] > [!UICONTROL Salvar como] padrão do Excel, como uma caixa de texto para entrada do nome de arquivo do relatório da planilha, usando a extensão de arquivo convencional [!DNL .xls].

Qualquer nome de arquivo usado precisa conter no máximo 255 caracteres. Além disso, o nome do arquivo não pode conter os seguintes caracteres:

\ ? | > &lt; : / &#42; &#39; &quot;

Finalmente, não é possível usar caracteres Unicode além do conjunto de caracteres ASCII estendidos.

Ao salvar o arquivo em um local nas suas unidades locais ou da rede, você poderá inserir o caminho completo na caixa de texto ou clicar no botão de navegação  ![browse_button.gif](assets/browse_button.gif), adjacente à caixa de texto [!UICONTROL Salvar como].
