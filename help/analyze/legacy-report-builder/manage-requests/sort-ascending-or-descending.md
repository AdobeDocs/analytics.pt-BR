# Classificação crescente ou decrescente

{{legacy-arb}}

Você pode classificar as solicitações listadas no Gerenciador de solicitações em ordem crescente ou decrescente.

Clique nos nomes das colunas no cabeçalho da tabela. A coluna mostra a ordem de classificação exibindo um ícone de triângulo no cabeçalho da tabela. Se o triângulo estiver apontando para baixo, a ordem de classificação será decrescente (para dados alfanuméricos, 9-1 e Z-A); se estiver apontando para cima, a ordem de classificação será crescente (para dados alfanuméricos, 1-9 e A-Z).

A ordem de classificação padrão é crescente.
