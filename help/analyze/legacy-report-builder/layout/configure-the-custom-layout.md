# Configurar o layout personalizado

{{legacy-arb}}

O layout personalizado oferece a maior parte das funcionalidades do layout dinâmico, mas permite que você escolha onde cada item na grade deve estar localizado na planilha. Esse layout oferece a flexibilidade disponível em versões anteriores, incluindo a adição de detalhamentos e micrográficos.

Os seguintes tópicos são exclusivos para o [!UICONTROL Layout Personalizado]:

* [Mapear métricas e dimensões para células](/help/analyze/legacy-report-builder/layout/map-metrics-and-dimensions-to-cells.md)
* [Criar um micrográfico](/help/analyze/legacy-report-builder/layout/t-create-a-microchart.md)
