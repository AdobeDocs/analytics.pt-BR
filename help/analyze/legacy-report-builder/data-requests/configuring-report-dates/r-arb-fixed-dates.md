# Definições do Assistente de solicitações - datas fixas

{{legacy-arb}}

Definições para datas fixas no Report Builder.

| Campo | Definição |
|--- |--- |
| Este conjunto de relatórios | Disponibiliza o intervalo de datas salvo somente para o conjunto de relatórios atual. |
| Ativando o Conjunto de relatórios | Disponibiliza o intervalo de datas salvo para todos os conjuntos de relatórios acessíveis ao usuário conectado. |
| Inserir nome do parâmetro | Digite um nome para a data salva. Você pode usar essa data como um modelo no Report Builder. Esse nome será exibido sempre que você puder aplicar uma data salva. A alteração do nome de um modelo não afeta a configuração de data usada em solicitações de dados criadas anteriormente. |
