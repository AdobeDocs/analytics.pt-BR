# [!UICONTROL Mapas de árvore]

Exibe dados hierárquicos (estruturados em formato de árvore) como um conjunto de retângulos aninhados.

Aqui está um vídeo sobre o [!UICONTROL Treemap] visualização:

>[!VIDEO](https://video.tv.adobe.com/v/334458/?quality=12)

Cada galho da árvore recebe um retângulo, que é empilhado com retângulos menores representando os sub-galhos.

![](assets/treemap.png)

Quando as dimensões de cor e tamanho estão correlacionadas de alguma forma com a estrutura de árvore, é possível, frequentemente, ver padrões que seriam difíceis de enxergar de outras formas, como se determinada cor é particularmente relevante. Uma segunda vantagem dos Treemaps é que, por construção, eles fazem uso eficiente do espaço.
