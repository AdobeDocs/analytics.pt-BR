# Implementar o Activity Map

O Activity Map pode ser implementado com uma caixa de seleção na extensão Adobe Experience Platform Launch para Adobe Analytics, em [Rastreamento de link](https://experienceleague.adobe.com/docs/launch/using/extensions-ref/adobe-extension/analytics-extension/overview.html?lang=en#link-tracking).
